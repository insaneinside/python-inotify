python-inotify
==============
python binding for linux inotify 

* License     :MIT

## Dependencies
* kernel >= 2.6.13
* cPython >= 2.7

## Install
### Clone this repo and install it with `python setup.py install`

    #git clone https://github.com/maliubiao/python-inotify.git
    $cd python-inotify
    $sudo python setup.py install
